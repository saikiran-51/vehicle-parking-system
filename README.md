# Vehicle Parking Management System 🚗

A full-stack Vehicle Parking Management System developed using React.js, Express.js, SQLite, and JWT Authentication.

## Features

- User Login Authentication
- Protected Routes
- Dashboard Analytics
- Parking Slot Management
- Slot Reservation System
- Reservation Cancellation
- Payment Popup Simulation
- Search & Filter Functionality
- Toast Notifications
- Responsive Modern UI

---

## Technologies Used

### Frontend

- React.js
- React Router DOM v5
- Axios
- React Toastify
- Reactjs Popup
- CSS

### Backend

- Node.js
- Express.js
- SQLite
- JWT Authentication

## Installation & Setup

### Clone Repository

```bash
git clone YOUR_GITHUB_REPO_LINK
```

---

### Frontend Setup

```bash
cd client
npm install
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```

---

### Backend Setup

```bash
cd server
npm install
node server.js
```

Backend runs on:

```bash
http://localhost:5000
```
